# star-wars-top-trumps

## Requirements:
- Node.js (~v12.4.1)
- NPM (~v6.14.3)

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Run your end-to-end tests (Cypress - Chrome browser required)
```
npm run test:e2e
```

### Lints and fixes files
```
npm run lint
```
